toggle flashlight to toggle battle mode
left/right arrows to navigate menu
down to enable action
up to disable action

in console: (if the battle mode breaks, use any of these)
pac_event mus_battle
	set battle type to default
pac_event mus_jevil
	set battle type to jevil
pac_event mus_hide
	sets battle type to none (hides music, menu, and stage)
pac_event reset
	use if the menu breaks somehow
